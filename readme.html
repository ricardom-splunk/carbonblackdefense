There are two different sets of credentials for this app - a SIEM key and an API key. The action get notifications uses the SIEM key. This means the siem_connector_id and the siem_key asset configuration parameters are required to run the get notifications action. All other actions require use the API key, meaning that the api_connector_id and api_key asset configuration parameters are required for those actions.

NOTE: Test connectivity will only check the API credentials, it will NOT check the SIEM credentials
